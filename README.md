**Stable after Phase 1 refactoring:**
- Successfully split the original `AgentMemory` god class
- Pure state + persistence now lives in `state.py`
- Core is now a thin orchestrator
- Windows sandbox + hardened `run_bash` tool working
- All daemons running, Obsidian wiki active, journals flowing to Hall of Records

## Core Features

- **Hybrid Memory**: Flat index + Chroma vector + Graph + Spatial OmniPalace (method of loci)
- **Living Obsidian Vault**: Auto-compile, self-healing wiki
- **Cerberus**: Multi-agent reasoning system
- **Claw**: Safe autonomous self-improvement daemon (proposes patches)
- **Autonomous Daemons**: AutoDream, Proactive Scheduler, Background Saver
- **XP & Personality**: Evolving traits based on feedback
- **Tool Safety**: Sandboxed file/bash operations with manual confirmation for edits
